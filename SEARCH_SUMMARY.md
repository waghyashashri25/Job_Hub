User types search term\n   ↓\n2. System normalizes & maps keyword\n   Example: \"intern\" → \"internship\"\n   ↓\n3. Flexible database search\n   - Matches: title, description, company\n   - Results ordered: latest first\n   ↓\n4. No results?\n   - Try without location filter\n   - Still nothing? Show latest jobs\n   ↓\n5. Display results + smart suggestion\n`\n\n### Generic Keyword Mapping\n`java\n\"intern\"      → Search \"internship\" category\n\"fresher\"     → Search \"entry-level, junior, beginner\"\n\"developer\"   → Search \"dev, engineer, programmer\"\n\"remote\"      → Search \"remote, work from home\"\n\"job\"         → Return latest jobs (all types)\n`\n\n---\n\n## Testing Quick Guide\n\n### Test These Searches (in order)\n1. **\"job\"** -
